# Web template for web / Сборщик веб проекта  
### Используемые библиотеки  
>Bootstrap 4.3.1  
>Owl-Carousel  
>Font-Awesome 4.7  
>jQuery  

### Используемые таски сборщика  
>Минификация js css, вывод в 1 файл  
>Препроцессор SASS  
>Минификация изображений  
>Автопрефиксер(кроссбраузерность)    

##### install
> npm i gulp@3.9.1

##### usage
> gulp

##### build for production
> gulp build
